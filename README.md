# AutoVision Analytics

An end-to-end **Data Analytics** project that analyzes automotive data to uncover valuable business insights related to vehicle pricing, engine performance, fuel efficiency, and market segmentation.

The project demonstrates the complete analytics workflow, including data cleaning, SQL-based business analysis, exploratory data analysis (EDA), and an interactive Power BI dashboard.

## Project Overview

AutoVision Analytics helps users explore automotive data through interactive visualizations and business-focused analysis. The project identifies the factors influencing car prices and compares vehicle performance across different manufacturers, fuel systems, engine locations, and body types.


## Objectives

- Analyze factors affecting vehicle prices.
- Compare vehicle performance across different brands.
- Evaluate fuel efficiency and horsepower.
- Identify market trends using SQL business queries.
- Build an interactive Power BI dashboard for decision-making.

## Business Questions Answered

- Which car brands have the highest average selling price?
- Which vehicles provide the best value for money?
- Does horsepower significantly influence vehicle price?
- Which fuel system is associated with higher-priced vehicles?
- Which engine location is more common?
- Which body type dominates the market?
- Which brands provide better fuel efficiency?
- How does average horsepower vary across manufacturers?
- What is the distribution of fuel systems?
- Which factors contribute most to premium vehicle pricing?

## Key Insights

- Premium brands generally command significantly higher average prices.
- Higher horsepower vehicles tend to have higher selling prices.
- Gas-powered vehicles dominate the dataset.
- Front-engine vehicles represent the majority of cars.
- Fuel efficiency varies considerably across manufacturers.
- Hatchbacks and sedans make up a large share of the available vehicles.
